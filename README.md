# Dự án Gợi ý Thực đơn Dinh dưỡng Cá nhân hóa

## Giới thiệu
Dự án sử dụng Python để phân tích dữ liệu dinh dưỡng và đưa ra thực đơn phù hợp dựa trên chỉ số cơ thể.

## Trạng thái dự án
- [x] **Giai đoạn 1:** Làm sạch và chuẩn hóa dữ liệu dinh dưỡng (Hoàn thành).
- [ ] **Giai đoạn 2:** Tính toán nhu cầu dinh dưỡng cá nhân (BMR, TDEE).
- [ ] **Giai đoạn 3:** Xây dựng thuật toán gợi ý món ăn.

## Công nghệ sử dụng
- Ngôn ngữ: Python
- Thư viện: Pandas, Numpy
- Công cụ: Google Colab
